## Prism
取名灵感来自美国的棱镜计划，`Prism`是一个采集用户性能的`SDK`，具体功能可以见功能列表介绍

## 功能列表  
- [x] 内存数据收集  
- [x] `cpu`数据收集  
- [x] `crash`数据收集  
- [x] 采集数据跟`Activity`生命周期绑定  
- [x] 内存泄漏处理能力
- [ ] `fps`监控（监听`touchUp`跟`touchDown`事件）
- [ ] 启动时间数据收集
- [ ] 页面加载时间数据收集
- [ ] 性能数据实时展示
- [ ] 设备基础性能收集（设备得分/设备`RAM`等）
- [ ] 监控数据统一处理（`kibana`/`acra`）
